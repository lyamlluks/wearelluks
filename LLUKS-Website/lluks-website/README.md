# LLUKS Website

A responsive one-page website for LLUKS.

## Files
- `index.html` — page structure and website copy
- `styles.css` — all styling and responsive layouts
- `script.js` — mobile menu, scroll animations, and footer date

## How to preview
Open `index.html` in a web browser.

## Before publishing
1. Replace the Instagram and Facebook `#` links in the footer.
2. Confirm the email address `hello@wearelluks.com`.
3. Replace the two Unsplash image URLs with your own branded photos when ready.
4. Upload all three website files to your host, keeping them in the same folder.

## Cloudflare Pages
You can drag the entire folder into Cloudflare Pages using Direct Upload, or connect a GitHub repository containing these files.
